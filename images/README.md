# Bilderordner für ROKKO! Records Homepage

Hier gehören alle Bilder für die Website hin.

## Benötigte Bilder

### Artist Porträts (8 Bilder)
- `skaramush-vandango.png` - SkaRamush Vandango Porträt
- `silberstreif.png` - Silberstreif Porträt
- `anger-uschis.png` - Anger Uschis Porträt
- `henri-bellieu.png` - Henri Bellieu Porträt
- `fleur-et-beunie.png` - Fléur et Beunié Porträt
- `sukram.png` - SUKRAM Porträt
- `anthony-sinclair.png` - Anthony Sinclair Porträt
- `rokko-logo.png` - ROKKO! Records Logo (wird als 8. Artist Card angezeigt)

### Header Bilder (1 Bild)
- `header-placeholder.png` - Header Image mit Logo und Künstler

## Anforderungen

### Größe und Format
- **Format:** PNG oder JPG/JPEG
- **Auflösung:** Min. 400x400px (optimal 600x600px)
- **Seitenverhältnis:** Quadratisch (1:1)

### Qualität
- Hochauflösend für HD-Displays
- Gut optimiert (unter 500KB pro Bild)
- Keine EXIF-Daten notwendig

## Speicherort

Alle Bilder müssen in diesem `/images` Ordner abgelegt werden:

```
images/
├── skaramush-vandango.png
├── silberstreif.png
├── anger-uschis.png
├── henri-bellieu.png
├── fleur-et-beunie.png
├── sukram.png
├── anthony-sinclair.png
├── rokko-logo.png
└── header-placeholder.png
```

## Bearbeitung

Die Bilder werden auf der Website automatisch:
- Mit 5pt Orange Border umrahmt
- Mit Hover-Zoom Effekt angezeigt
- Responsive skaliert (passt sich Bildschirmgröße an)
- Mit abgeschrägten Label versehen

## Design-Hinweise

- Dark Mode Background: Die Website hat schwarzen Hintergrund
- Orange Akzente: Bilder werden mit Orange (#E7760D) gerahmt
- Kontrast: Stelle sicher, dass die Künstler/Personen gut sichtbar sind
- Dunkelheit: Bilder mit dunklem Hintergrund funktionieren am besten
